# localizer

## Requirement
- ros (kinetic)
- PCL 1.8

### Subscrived topics
- /odom (nav_msgs/Odometry)
- /imu/data (sensor_msgs/Imu)
- /velodyne_points(sensor_msgs/PointCloud2)

- run 
```
~$  ./run.sh
```
